## 📌 Overview

NayePankh AI Hub is an AI-powered platform designed to support NGOs and social organizations in managing volunteers, analyzing campaigns, automating workflows, and making data-driven decisions.

The platform demonstrates the practical application of Artificial Intelligence, Machine Learning, Data Analytics, and Automation in the social impact sector.

